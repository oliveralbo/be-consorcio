## Qué hace
Carga los archivos de contexto relevantes para trabajar con la API del proyecto be-consorcio.

## Cuándo usarlo
Usa este skill cuando:
- Necesites crear o modificar endpoints
- Trabajes con autenticación o guards
- Implementes lógica de negocio (expensas, gastos, dashboard)
- Debuguees flujos de la API

## Archivos que carga
Lee los siguientes archivos en orden:
1. `.opencode/context/02-architecture.md` — Arquitectura de módulos
2. `.opencode/context/04-api-endpoints.md` — Endpoints de la API
3. `.opencode/context/05-auth-and-guards.md` — Sistema de autenticación
4. `.opencode/context/06-business-logic.md` — Lógica de negocio clave
